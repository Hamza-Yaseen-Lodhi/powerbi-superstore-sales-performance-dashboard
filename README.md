# Superstore Sales Performance Dashboard using Power BI

## Project Overview

This project is an interactive Power BI dashboard built to analyze Superstore sales performance across sales, profit, orders, customers, product categories, regions, discounts, and payment modes.

The main goal of this dashboard is to convert raw sales data into clear business insights using KPI cards, slicers, trend charts, category analysis, regional analysis, and payment-mode visuals.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel / CSV Dataset

## Dashboard Pages

### 1. Sales Report Dashboard
This page provides an executive overview of business performance, including total sales, profit, orders, customers, discounts, monthly trends, region-wise sales, category analysis, and payment-mode performance.

### 2. Sample Sales Dashboard
This page provides additional analysis using visuals such as gauge charts, category treemaps, city-wise sales, order trends, sales by region, and sales vs orders comparison.

## Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Average Discount
- Average Order Value
- Profit Margin
- Total Quantity

## Dashboard Features

- KPI cards for quick business overview
- Interactive slicers for filtering data
- Monthly sales trend analysis
- Sales vs profit comparison
- Category-wise sales analysis
- Product-level sales analysis
- Region-wise sales analysis
- Payment mode breakdown
- Monthly performance table
- Map-based regional visualization

## Business Insights Covered

This dashboard helps answer questions such as:

- What is the overall sales and profit performance?
- Which regions are generating the most sales?
- Which product categories are performing best?
- How do sales and profit change over time?
- Which payment modes are used most frequently?
- Which products and categories contribute most to revenue?
- How do discounts affect business performance?
